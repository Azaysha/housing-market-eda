# 🏠 Housing Market — Exploratory Data Analysis

## 📌 Project Overview
Performed Exploratory Data Analysis (EDA) on a housing dataset 
to identify key factors that influence property sale prices 
using statistical methods.

> 📂 Dataset provided by Professor Henrique Ewbank  
> 🎓 ADTA 5130 — Data Analytics 1 | University of North Texas

## 📊 Dataset
- 19,432 housing transactions
- 16 variables (price, bedrooms, bathrooms, sqft, condition)
- Source: Provided by course instructor

## 🛠️ Tools Used
- Microsoft Excel
- ANOVA & Tukey HSD Post-Hoc Test
- Simple Linear Regression

## 🔍 Key Findings
- sqft_living is strongest price predictor (R² = 49.6%)
- Better property condition = higher sale price
- Each bathroom adds ~$254K to price on average
